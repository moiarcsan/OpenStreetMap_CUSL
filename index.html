Configurar Repo

Antes de crear las transparencias debemos crearnos un repo en local para poder trabajar. Los pasos que se explican aquí es para un entorno Linux

Configurar Repo

Git Clone

git clone url_repo

Git Fetch

git fetch origin

Git Checkout

git checkout gh-pages

Git Submodule

git submodule add https://github.com/hakimel/reveal.js.git revealjs
git submodule init
git submodule update

Visualización

Para poder ver cómo va quedando nuestras transparencias tenemos que configurar nuestro entorno.
Nos creamos un enlace simbólico en /var/www con el comando ln -s carpeta_origen destino
Todos los cambios que vayamos haciendo se verán en http://localhost/destino
